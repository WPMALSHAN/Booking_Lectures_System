## 📌 Project Overview

The **Online Lecture Booking System** is a full-stack web application that allows students to book appointments with lecturers in an efficient and organized way.

It provides a structured academic scheduling platform with **role-based access control** for Students, Lecturers, and Admins.

---

## 🚀 Features

### 👨‍🎓 Student
- Register & login securely
- View lecturers
- Book lecture appointments
- View booking history
- Submit feedback

### 👨‍🏫 Lecturer
- Set availability schedule
- Manage student appointments
- Accept / reject bookings
- View student feedback

### 🧑‍💼 Admin
- Manage users (students & lecturers)
- Block / unblock users
- Monitor system activity
- Manage system data
